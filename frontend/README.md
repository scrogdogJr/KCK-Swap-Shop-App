# Frontend

React app for KCK Swap Shop

## Setup

Install dependencies:
```bash
npm install
```

Start development server:
```bash
npm start
```

Build for production:
```bash
npm run build
```

## Project Structure

- `public/` - Static HTML
- `src/` - React components and pages
  - `components/` - Reusable components (Navigation, etc)
  - `pages/` - Page components (Home, etc)
